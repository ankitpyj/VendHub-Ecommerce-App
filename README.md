# 🛒 VendHub Ecommerce App

## 📌 Overview
VendHub is a full-stack ecommerce platform designed to directly connect **farmers and vendors**, eliminating middlemen and enabling fair trade.

This application allows farmers to list their products and vendors to purchase them directly, creating a transparent and efficient supply chain.

---

## 🌾 Problem Statement
Traditional supply chains often involve multiple intermediaries, reducing profits for farmers and increasing costs for vendors.

👉 VendHub solves this by:
- Direct farmer-to-vendor connection  
- Transparent pricing  
- Efficient product management  

---

## 👥 User Roles

### 👨‍🌾 Farmer
- Add and manage products  
- Update product details (price, quantity, etc.)  
- View orders from vendors  

### 🏪 Vendor
- Browse available products  
- Add products to cart  
- Place orders directly from farmers  

### 🛠 Admin
- Manage users (farmers & vendors)  
- Monitor platform activity  
- Handle product approvals
